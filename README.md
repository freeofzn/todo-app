# 📝 Todo App

간단하고 깔끔한 Todo 앱입니다. Vanilla JavaScript로 만들어졌으며, 기본적인 CRUD 기능을 제공합니다.

## 🎯 기능

- ✅ **할 일 추가** - 새로운 할 일을 입력하고 추가
- ✅ **할 일 완료 표시** - 체크박스로 완료 상태 관리
- ✅ **할 일 삭제** - 불필요한 할 일 삭제
- ✅ **필터링** - 전체/진행중/완료됨으로 필터링
- ✅ **통계** - 전체 할 일 수와 완료 수 표시
- ✅ **데이터 저장** - 로컬 스토리지에 자동 저장

## 🚀 사용 방법

### 1. 저장소 클론
```bash
git clone https://github.com/freeofzn/todo-app.git
cd todo-app
```

### 2. 브라우저에서 열기
- `index.html` 파일을 브라우저에서 열면 됩니다.
- 또는 로컬 서버를 사용하여 열기:
```bash
# Python 3
python -m http.server 8000

# Node.js (http-server 설치 필요)
npx http-server
```

### 3. 사용하기
1. 입력 필드에 할 일을 입력
2. "추가" 버튼을 클릭하거나 엔터 키 누르기
3. 체크박스로 할 일 완료 표시
4. "삭제" ���튼으로 할 일 제거
5. 필터 버튼으로 상태별 보기

## 🛠 기술 스택

- **HTML5** - 마크업
- **CSS3** - 스타일링 (그래디언트, 플렉스박스, 반응형 디자인)
- **Vanilla JavaScript** - 기능 구현 (DOM 조작, 로컬 스토리지)

## 💾 데이터 저장

- 모든 데이터는 브라우저의 **로컬 스토리지**에 저장됩니다.
- 브라우저를 종료해도 데이터가 유지됩니다.
- 캐시를 삭제하면 데이터가 초기화됩니다.
